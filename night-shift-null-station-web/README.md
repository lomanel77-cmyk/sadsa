# Night Shift: Null Station

Originální 3D survival-horor pro webový prohlížeč. Projekt nepoužívá postavy, mapu, zvuky ani grafické materiály z Five Nights at Freddy's. Všechny modely, textury a zvuky vznikají procedurálně v kódu.

## Ovládání

- Myš nebo A / D: rozhlížení
- C: bezpečnostní kamery
- 1–8: přepínání kamer
- Z / X: levé dveře / světlo
- N / M: pravé dveře / světlo
- V: uzávěr ventilace
- P: pauza

## Herní systémy

- 8 bezpečnostních kamer
- 3 nepřátelské jednotky s odlišnou AI
- Levé a pravé bezpečnostní dveře
- Ventilační uzávěr
- Omezená energie a dynamická spotřeba
- Výpadek proudu, rušení kamer a detekce pohybu
- Procedurální Web Audio zvuky
- Tři obtížnosti a automatická kvalita grafiky
- Ovládání pro počítač i dotyková zařízení
